# How to Play Minecraft Java Edition 1.12.2

This guide explains how to play **Minecraft 1.12.2** on PC, Chromebook, and in web browsers using Eaglercraft.

---

## 1. Playing Official Java Edition (PC / Mac / Linux)

1. **Install the Minecraft Launcher**  
   Download the official launcher from [minecraft.net](https://www.minecraft.net/en-us/download) and install it.

2. **Login**  
   Use your **Mojang or Microsoft account** to log in.

3. **Select Version**  
   - In the launcher, go to **Installations → New → Version**  
   - Choose **1.12.2** from the version dropdown  
   - Save and click **Play**

4. **Start Playing**  
   - Create a new world or join a multiplayer server  
   - Enjoy all blocks, mobs, and features available in 1.12.2

---

## 2. Playing in Eaglercraft (Chromebook / Browser)

Chromebooks and some PCs cannot run Java Edition natively, but you can play **Minecraft 1.12.2** in your browser using **Eaglercraft**:

### Steps:

1. Open the Eaglercraft website: [https://eaglercraft.dev/](https://eaglercraft.dev/)  
2. Click **Play** on the main page.  
3. Select **Minecraft 1.12.2** from the version dropdown.  
4. The game will load in your browser using WebGL.  
5. Use the keyboard and mouse normally to play.

> ⚠️ Note: Performance depends on your device. Older Chromebooks may experience lag.  
> ⚠️ Some mods or resource packs may not be supported in browser builds.

### Tips for Chromebook:

- Use **fullscreen mode** (F11) for the best experience.  
- Close unnecessary tabs to improve performance.  
- Eaglercraft saves progress in browser storage — do not clear cache to keep your worlds.

---

## 3. Installing Mods

1. **Install Forge 1.12.2**  
   - Download Forge from [https://files.minecraftforge.net/](https://files.minecraftforge.net/)  
   - Install using the installer for **1.12.2**

2. **Add Mods**  
   - Place `.jar` files into the `mods` folder of your Minecraft installation  
   - Launch the game using the Forge profile

3. **Enjoy Modded Gameplay**  
   - Many servers and custom maps require Forge 1.12.2  
   - Always download mods from trusted sources

---

## 4. Multiplayer Servers

- Minecraft 1.12.2 servers are still active, especially modded servers  
- Join using **direct IP** or via server lists online  
- Some browser ports like Eaglercraft allow joining public servers directly in-browser

---

## 5. Tips & Recommendations

- Keep **Java updated** (Java 8 recommended)  
- Allocate more RAM in launcher settings if running mods  
- Backup your worlds before installing mods  
- For Chromebooks or low-end PCs, prefer **Eaglercraft** to avoid performance issues  
- Always use **official or trusted sources** for downloads and mods
